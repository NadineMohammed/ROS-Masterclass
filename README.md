## 🎯 What I'll Learn

- Python fundamentals for robotics (OOP, loops, functions)
- Linux terminal and ROS2 packages & workspaces
- Publisher/Subscriber, Services, and Actions patterns
- Robot modeling with URDF (links, joints, geometry)
- TF2 transformations and coordinate frames
- Map building using SLAM Toolbox and 2D LiDAR
- Robot localization using AMCL particle filter
- Full autonomous navigation with the Nav2 stack
